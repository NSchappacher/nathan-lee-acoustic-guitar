# Nathan Lee Acoustic Guitar — Single Page Site

A rustic, responsive one-page website for booking, contact, reviews, and gratuity.

## How to Deploy (GitHub Pages — Free)
1. Create a new GitHub repository (e.g., `nathan-lee-acoustic-guitar`).
2. Add files/folders as shown in the structure above (use the web editor or git).
3. Go to **Settings → Pages**:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/root`
4. Click **Save**. Your site will be live at `https://YOUR_USERNAME.github.io/REPO_NAME`.

## Images
Place these files in `assets/images/`:
- `hero.jpg` — shows in the top hero section
- `about.jpg` — beside the About section
- `contact.jpg` — near Contact Information
- (Optional gallery) `gallery-1.jpg` … `gallery-6.jpg`

## Reviews Form → Email
The form posts to **FormSubmit** and forwards to `nschappy1@gmail.com`. Edit the address if needed in `index.html`.

## Customize
- Colors & fonts: `css/styles.css` (`:root` variables).
- Content: `index.html`.
- Favicon: `assets/icons/favicon.svg`.
